
GitHub Pages 部署包 — 给雨汐的告白（增强版）

功能概览：
- 更自然的彼岸花飘落（较少、缓慢、轻飘）
- 背景支持：把 background.jpg / background.png 放在仓库根目录并在页面输入文件名，或通过页面上传本地背景（仅会话内生效）
- 新的浪漫中国风诗句，突出“你是我的唯一”
- 音乐歌词同步（LRC 支持）：可上传 .lrc 文件或把 lyrics.lrc 放在仓库并在页面输入文件名
- 相册仅会话预览（避免 localStorage 容量问题）

部署：将此文件夹的内容上传到 GitHub 仓库并启用 Pages（Settings → Pages → main branch / root），几分钟后访问 https://<你的用户名>.github.io/<仓库名>/

说明：
- 请将你想要的背景图放在仓库根目录并在首页输入文件名，例如 background.jpg，然后点击“使用此背景”。或者直接点击上传（bg upload）在手机/电脑会话中即时显示。
- 歌词格式为 LRC（示例：[00:12.34]歌词文本），页面会自动解析时间并高亮滚动。
