
Confession page (poem + lyrics) — package info

Files:
- index.html
- style.css
- script.js
- background.jpg (your central image; replace with your own)
- music.mp3 (optional)
- lyrics.lrc (LRC file, included)

Usage:
1. Upload all files to your GitHub Pages repo root (or open index.html locally).
2. Open the page. The background image will appear center; poem types out below the image per character with rainbow gradient.
3. Click '播放/暂停' to play music; lyrics will sync and highlight current line and center it in the lyrics box.
